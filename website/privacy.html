Privacy

keephippo.com is a static website hosted on GitHub Pages. It sets no cookies, runs no analytics or trackers, and collects no personal information. The only network requests the pages make are:

  • loading the site's own assets (HTML, CSS, JS, images) from this domain, and
  • on the Downloads page, a request to the public GitHub Releases data (served from a static versions.json generated at deploy time) to list download links.

GitHub, as the host, may collect standard server logs (such as IP addresses) per its own privacy statement. The keephippo software itself — the server and CLI you download and run — makes no outbound connections and phones nothing home; your secrets never leave your machine.

Questions? Open an issue.
